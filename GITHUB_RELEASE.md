# GitHub release draft

## Title

v0.1.0 - first public release

## Summary

Initial public release of `agent-skill-kit`, a tiny CLI for scaffolding and linting AI agent skills.

## Included in this release

- `askit init <name>` to scaffold a new skill
- `askit lint [path]` to validate one skill
- `askit lint-all [path]` to scan a whole repo
- `askit readme [path]` to generate a starter README
- `askit checklist` to run a pre-publish checklist

## Why it matters

Most skill repos still rely on copy-paste setup and inconsistent documentation. This release is a small but practical starting point for making skill authoring cleaner and more repeatable.
